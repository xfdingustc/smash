PC Host App for UsbVscDemo

Overview
==========
PC Host app for USB VSC, using libusb.

Build
==================
To build the project please type:
"make clean"
"make"

Setup
==================
  - connect a USB3.0 or USB2.0 cable from MV182 or MV212 board to the a PC
  - open terminal and type "sudo ./vscPC"

Expected output
==================
Read/Wrote x bytes, x representing transfer size
